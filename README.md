## Getting Started

### Prerequisites

```bash
Node.js (v14 or later)
pnpm
```

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/tracytratran/tracytratran.github.io.git
   cd tracytratran.github.io
   ```
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Start the development server:
   ```bash
   pnpm start
   ```
4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
